This lz4 addon requires lz4 binary library to encode and decode
lz4 raw and frame blocks.

For Linux, lz4 can be installed by using

# Debian derivatives
$ sudo apt-get install lz4

# Redhat derivatives
$ sudo dnf install lz4

For Mac, lz4 can be installed from homebrew

$ brew install lz4

For Windows, type the following in a J session to download and install
lz4 dll from Jsoftware server.

load 'arc/lz4'
install_jlz4_''
load 'arc/lz4'
